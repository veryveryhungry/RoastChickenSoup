# RoastChickenSoup
회원조회 및 게시판

## BackEnd
스프링과 MySQL을 이용하여 '소박하지만 그룹입니다' 홈페이지 회원 목록 및 조회, CRUD 기능을 담당하는 게시판을 만들 목적으로 
개설된 리포지토리입니다.
